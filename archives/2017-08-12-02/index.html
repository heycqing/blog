关于这个博客

2017-08-12

好吧….之前想做自己的博客很久了，本来会想做一个完全由自己搭建的独立博客，好让自己学到的一些东西可以搭建成一个独立的博客，但是由于种种原因，博客迟迟未搭建起来。

嗯…..然后平时自己有收藏一些关于编程的知识或者自己的一些看法，但是都没有一个地方去展示，所以就用了 wordpress 这个博客模板来搭建。

然后呢，个人觉得 wordpress 这个博客模板，是基于 php 写的，但是我php学得不深，不好做评论，但是看了 wordpress 的 css 源码，觉得有一些地方还有待改进，然后呢又觉得这样可以自己去编写一些css样式去改善一下外观，也是对自己的编程的一个锻炼，所以就选择在这个时候，搭建了自己博客。 在这个博客上面我发一些我过去0自己的经验，和实战，还有一些个人对前端的理解。

然后，往后在这个博客上，我也会分享一些自己编程上的一些经验，一些有助提升自己的知识，一些自己的生活和想法。

好吧，这个博客第一篇就这样吧。

写于用 wordpress 创建的博客，现在迁移到 hexo 上了。
