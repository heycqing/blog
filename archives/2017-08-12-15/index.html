阿里云服务器搭建

2017-08-12

教程之前的声明；

之前有使用过不同产商的服务器，

  1. 有新浪云的，这个要冲云豆，但是里面是一套配好的系统；
  2. 有腾讯云的，作为云服务器，这个众所周知，都要自己去配置环境的；
  3. 有阿里云的，和腾讯云一样的，要自己去配好环境，作为学生，当然选择了阿里云这个服务商，比较有优惠。可以点击来获取优惠卷；https://promotion.aliyun.com/ntms/act/ambassador/sharetouser.html?userCode=j8vnzlte&productCode=vm&utm_source=j8vnzlte

下面是搭建教程：

在阿里云购买服务器后选择要安装的系统；

我选择的是CentOS 7.3 64位的；
内置的php版本是5.多6.多
如果使用ubuntu则是7.多；

步骤1：

我需要在与服务连接 xshell 应用来在桌面使用命令行；

可以在Xshell 上使用 ssh root@主机名连接，输入用户名和密码；

步骤2：

需要升级一下apt,不然无法从网上获取响应的模块；

用到命令是：

apt update

升级完后，就可以对云服务器为所欲为了；

步骤3：

因为我需要安装mysql来接连 sql可视化软件来进行管理,然后呢,我用命令行：

apt install mysq-server

apt install mysql-client

mysql的c语言的开发接口；

apt install libmysqlclinet15-dev

然后或出现一系列的安装信息,输入用户名和密码即可,安装完后可以使用

netstat -tap | grep mysql

来检查 mysql 是否处于 listen 状态,然后，就是使用 mysql 了；

mysql -u 用户名 -p密码

就会出现下面的图示

可以使用下面这个来查看数据库；

show databases;

在mysql终端里，记得在命令结束后加上一个;

在与服务器里安装好 mysql 后，我想要连接本机电脑上的可视化sql软件,然后,出现了下面的错误，无法连接上；

是因为要在云服务器上做好配置,首先要在云服务器的 mysql 分配好可以连接远程主机的配置；

然后mysql>exit; 退出 mysql

然后vim /etc/mysql/my.cnf

然后就可以连上了；

步骤4：

我需要安装 一个代理服务器 nginx

apt install nginx

然后可以使用

netstat -tap | grep nginx

查看当前 nginx 的状态；

在浏览器里输入ip地址就可以访问了；
